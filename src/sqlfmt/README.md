
# sqlfmt (via pipx) (sqlfmt)

The autoformatter for dbt SQL

## Example Usage

```json
"features": {
    "ghcr.io/cedric-orange/devcontainer-features/sqlfmt:0": {
        "version": "latest"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | Select the version to install. | string | latest |
